User Upload

Please click the button below to download the csv template to be filled in with your custom information.
NOTE: Please fill in the template with the exact parameters of the csv template file.

(R) indicates required input.
(M) indicates for Merchants only.

If you have commas within the input, put quotation marks around entire string.
Also note that the Users created will have passwords identical to their Username/Email.
You may open the .csv file in any format to edit details.

WARNING:
Please make sure that there are all information given follow the exact parameters of the template.

Item Upload

Please click the button below to download the csv template to be filled in with your custom information.
NOTE: Please fill in the template with the exact parameters of the csv template file.

(R) indicates required input.

If you have commas within the input, put quotation marks around entire string.
If you have multiple inputs under the same section (i.e. Categories, tags, etc.),
separate by commas and put the entire string in quotation marks.
You may open the .csv file in any format to edit details.

THIS FUNCTION DOES NOT SUPPORT ITEMS WITH VARIANCE.

WARNING:
Please make sure that there are all information given follow the exact parameters of the template.
Items on your marketplace can be identical, so be careful not to upload the same file twice to avoid doubling.
Please upload your updated ItemTemplate.csv file below.
